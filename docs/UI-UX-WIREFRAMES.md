## 1. Design System Overview

### 1.1 Brand Colors

| Token           | Light Mode | Dark Mode | Usage                         |
| --------------- | ---------- | --------- | ----------------------------- |
| `primary`       | #DC2626    | #DC2626   | Primary actions, brand accent |
| `background`    | #FFFFFF    | #0A0A0A   | Page background               |
| `surface`       | #FEF2F2    | #262626   | Cards, elevated surfaces      |
| `text`          | #0A0A0A    | #FFFFFF   | Primary text                  |
| `textSecondary` | #525252    | #A3A3A3   | Secondary text                |
| `border`        | #FECACA    | #404040   | Borders, dividers             |

### 1.2 Typography Scale

| Style       | Size | Weight   | Line Height | Usage              |
| ----------- | ---- | -------- | ----------- | ------------------ |
| `h1`        | 32px | Bold     | 40px        | Page titles        |
| `h2`        | 24px | Bold     | 32px        | Section headers    |
| `h3`        | 20px | Semibold | 28px        | Card titles        |
| `h4`        | 18px | Semibold | 28px        | Subsection headers |
| `body`      | 16px | Regular  | 24px        | Body text          |
| `bodySmall` | 14px | Regular  | 20px        | Secondary content  |
| `caption`   | 12px | Regular  | 16px        | Labels, metadata   |

### 1.3 Spacing Scale

```
4px  (xs)   - Tight spacing, icon padding
8px  (sm)   - Button padding, inline gaps
16px (md)   - Card padding, section gaps
24px (lg)   - Major section spacing
32px (xl)   - Page margins, hero spacing
48px (xxl)  - Large section breaks
```

### 1.4 Existing Components

| Component   | Location                             | Variants                           |
| ----------- | ------------------------------------ | ---------------------------------- |
| `Button`    | `/components/Button.tsx`             | primary, secondary, outline, ghost |
| `Card`      | `/components/Card.tsx`               | default, elevated                  |
| `Input`     | `/components/Input.tsx`              | text, password, email              |
| `Text`      | `/components/typography/Text.tsx`    | body, bodySmall, bodyLarge         |
| `Heading`   | `/components/typography/Heading.tsx` | h1, h2, h3, h4                     |
| `Avatar`    | `/components/display/Avatar.tsx`     | sm, md, lg                         |
| `Badge`     | `/components/display/Badge.tsx`      | default, success, warning, error   |
| `Stack`     | `/components/layout/Stack.tsx`       | vertical, horizontal               |
| `Container` | `/components/layout/Container.tsx`   | responsive wrapper                 |

### 4.2 Component API Examples

#### ContentCard

```typescript
interface ContentCardProps {
  id: string;
  type: 'news' | 'article' | 'event' | 'newsletter';
  title: string;
  excerpt?: string;
  imageUrl?: string;
  publishedAt: Date;
  author?: { name: string; avatarUrl?: string };
  likesCount: number;
  commentsCount: number;
  isLiked?: boolean;
  onPress: () => void;
  onLike?: () => void;
}
```

#### ChatBubble

```typescript
interface ChatBubbleProps {
  role: 'user' | 'assistant' | 'system';
  content: string;
  timestamp: Date;
  isEscalation?: boolean;
  status?: 'sending' | 'sent' | 'error';
}
```

#### DataTable

```typescript
interface DataTableProps<T> {
  columns: Column<T>[];
  data: T[];
  sortable?: boolean;
  selectable?: boolean;
  onRowClick?: (row: T) => void;
  onSelectionChange?: (selected: T[]) => void;
  pagination?: {
    page: number;
    pageSize: number;
    total: number;
    onPageChange: (page: number) => void;
  };
}
```

## 5. Responsive Breakpoints

### 5.1 Breakpoint Scale

| Breakpoint | Width       | Device Target |
| ---------- | ----------- | ------------- |
| `xs`       | 0-479px     | Small phones  |
| `sm`       | 480-767px   | Large phones  |
| `md`       | 768-1023px  | Tablets       |
| `lg`       | 1024-1279px | Small laptops |

| Element       | Mobile (xs-sm)       | Tablet (md)          | Desktop (lg+) |
| ------------- | -------------------- | -------------------- | ------------- |

| Card size     | Full width           | 50% width            | 33% width     |
| Content width | 100%                 | 90%                  | max 1200px    |
| Font sizes    | Base scale           | Base scale           | +1 step       |
